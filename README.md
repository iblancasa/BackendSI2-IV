# SIMINTA
SIMINTA adalah template admin responsif yang menggunakan kerangka twitter Bootstrap. Admin SIMINTA dilengkapi dengan koleksi besar plugin dan komponen UI yang membuat pekerjaan Anda mudah. Semua kode yang ditulis pada halaman / file yang telah disiapkan sesuai kebutuhan Anda. Template ini dapat digunakan secara gratis untuk keperluan pribadi maupun komersial.
